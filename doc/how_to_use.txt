一、功能介绍
    FusionStudent Platform v5.0是一款教室-学生管理系统。可以通过该系统对教室和学生进行增、删、查询。
    系统可靠性优良，空间复杂度低，适合中小教育机构对课堂进行管理操作。
    教室拥有属性：id、名称、大小、备注、剩余空间大小（还可容纳的学生人数）、状态。
        id: 每个教室的唯一标识。在创建教室的过程中该属性由系统自动生成，且每个教室的id不会重复。
        名称：教室的名称。名称是可选的，可由用户指定。当用户不指定名称时，系统自动根据id生成名称。
        大小：教室座位大小。必须是整数，且必须指定。教室的大小必须是一个大于0的数字。
        备注：教室功能备注，可选选项。
        剩余空间大小：表示教室还可以容纳的学生人数。该数据是根据教室大小和教室存在的学生数量动态生成的，
            不由用户创建指定。
        状态：教室状态。目前支持状态如下：
            1. "Opening"。表示教室正在开放中。
            2. "Lock"。表示教室处于锁定状态。

    除基本功能外，FusionStudent Platform v5.0 还具有日志查询功能、日志清除功能以及系统时间查询、良
    好的系统关机指令、更换操作系统登录密码以及恢复出厂设置等功能。系统关机时，所有创建的数据不会丢失，
    系统会自动保存。


二、前置条件
    所运行的机器推荐安装Python3.6.5以及以上版本，至少保证在Python2.4版本及以上。

三、运行条件
    推荐运行平台Mac OS X
    推荐通过终端运行。进入FusionStudent目录后，使用命令：python platform.py即可运行系统。


四、功能
    1. 开机
    （功能已在FusionStudent Platform 3.2.6 版本验证通过。）
    运行../FusionStudent/platform.py，系统开机。显示效果如下：
    ➜  FusionStudent python platform.py
    # username:

    2. 登录
    （功能已在FusionStudent Platform 3.2.7 版本验证通过。）
    运行系统后，提示需要输入账号密码才能登录。
    账号：admin
    默认密码：123fs
    显示效果如下：
    # username: admin
    # password:
    Login system success.
    Welcome FusionStudent Platform system.
    FusionStudent Platform 3.2.7
    FusionStudent Platform System starting... please wait.
    FusionStudent Platform System start success.

    3. 查询日志：
    （功能已在FusionStudent Platform 1.7.5版本验证通过。）
    所有产生的日志均在../FusionStudent/log目录下。
    info.log记录日常操作；
    error.log记录产生的错误；
    warn.log记录产生的高危搞作。

    4. 命令帮助

            clear help:  clear resource operation to help.

            student help:  student operation to help.

            system reset password:  reset admin password.

        system version 显示当前版本。
        system exit 退出系统。
        system date 显示当前系统时间。
        system reset password 更换系统开机密码。

    clear help 显示clear命令集帮助。显示效果如下：

        clear system    恢复出厂设置。
        ⚠️该操作是危险操作，请谨慎使用。

    class help 显示class命令集帮助。显示效果如下：

    （功能已在FusionStudent Platform 1.7.7版本验证通过。）
    输入命令 system version 查询当前版本。显示效果如下：

    5. 退出系统（关机）
    （功能已在 FusionStudent Platform lite 4.6.6 版本验证通过。）
    A. 输入命令：system exit 回显如下：
        >>> system exit
        Are you sure about exit system?(YES/NO)
    B. 若不退出，继续返回系统，输入 NO。回显如下：
        Are you sure about exit system?(YES/NO) NO
        Welcome to back.
    C. 若仍选择退出，输入YES。回显如下：
        Are you sure about exit system?(YES/NO) YES
        Thanks.
        SHUTDOWN
        System run 104.33 seconds.
    ⚠️ 若不使用该命令强制退出，会导致系统无法使用正常命令再次登录。
      若意外情况导致此问题发生，按如下方式规避：
      打开../FusionStudent/conf/keypair.json
      将 login 对应的值修改为 false
      重新使用命令开机即可

    6. 查询系统时间
    （功能已在FusionStudent Platform 1.7.9版本验证通过。）
    输入命令 system date 回显如下：
    >>> system date
    2019/7/5 17:34:23 CST

    7. 清除日志功能
    （功能已在FusionStudent Platform lite 4.4.1 版本验证通过。）
    A. 若清除全部日志，输入命令 clear log --all 回显如下：
        >>> clear log --all
        Clear log success.
    B. 清除指定日志。例如清除info日志：
        >>> clear log --info
        Clear log success.
       清除error日志：
        >>> clear logs --error
        Clear log success.
       清除warn日志：
        >>> clear log --warn
        Clear log success.
       也可以组合使用：
        >>> clear log --info --error --warn
        Clear log success.

    8. 创建class
    （功能已在FusionStudent Platform 1.8.2版本验证通过。）
    输入命令 class create --size 教室容量(必选) --name 教室名称 --remark 教室说明
    例如：
    >>> class create --size 9 --name DEMO --remark DEMO_Class
    Class R1US8TDF-4UYK-QGZ8-NF6F-TKU60175 create success.
    新的教室(ID: R1US8TDF-4UYK-QGZ8-NF6F-TKU60175)创建成功。
    ⚠️：remark描述中，单词与单词之间不能有空格。

    9. 查看class列表
    （功能已在FusionStudent Platform lite 4.4.1版本验证通过。）
    输入命令 class list 查看已存在的教室列表。
    例如：
    >>> class list
    +----------------------------------+--------+
    | uuid                             | name   |
    +----------------------------------+--------+

    10. 查看class详细信息
    （功能已在FusionStudent Platform 3.3.0 版本验证通过。）
    输入命令 class show class_ID 查看class的详细信息。例如：
    >>> class show YR29XCJZ-EWB0-GK1U-K5TT-WHMQIO8T
    +---------+----------------------------------------
    | uuid:   | YR29XCJZ-EWB0-GK1U-K5TT-WHMQIO8T
    | name:   | test
    | size:   | 8
    | remark: | None
    | free:   | 8
    | status: | Opening
    +---------+----------------------------------------

    11. 删除class
    （功能已在FusionStudent Platform lite 4.4.3 版本验证通过。）
    输入命令 class delete class_ID 删除class。例如：
    >>> class delete R1US8TDF-4UYK-QGZ8-NF6F-TKU60175
    Delete class 'R1US8TDF-4UYK-QGZ8-NF6F-TKU60175' success.
    ⚠️ 当class内存在student实例时，class不能删除。
    ⚠️ 当class处于Lock状态时，class不能删除。

    12. 锁定class
    （功能已在FusionStudent Platform lite 4.5.1 版本验证通过。）
    输入命令 class lock <uuid> 锁定一个class
    >>> class lock 6MW24UVO-17TP-BX8G-ULFC-LKJUIUQP

    class锁定后，状态变更为Lock。不能删除、不能添加和删除该class下的学生，恢复出厂设置也会失败。
    ⚠️ 只有当class状态为Opening时才能执行此命令

    13. 解锁class
    （功能已在FusionStudent Platform lite 4.5.2 版本验证通过。）
    输入命令 class unlock <uuid> 解锁一个class
    >>> class unlock 6MW24UVO-17TP-BX8G-ULFC-LKJUIUQP

    class解锁后，状态恢复成Opening。
    ⚠️ 只有当class状态为Lock时才能执行此命令

    14. 创建student
    （功能已在FusionStudent Platform lite 4.4.4 版本验证通过。）
    >>> student create --name Bob --sex boy --class YR29XCJZ-EWB0-GK1U-K5TT-WHMQIO8T
    Student vm3f11js-6ggg-bg8h-urhw-2jfp179q create success.
    ⚠️ 当class的free小于或等于0时，student创建失败。

    （功能已在FusionStudent Platform lite 4.4.5 版本验证通过。）

    >>> student delete vm3f11js-6ggg-bg8h-urhw-2jfp179q

    ⚠️ 当删除一个不存在的学生uuid时，删除失败。

    18. 修改admin密码
    （功能已在 FusionStudent Platform lite 4.4.6 版本验证通过。）
    >>> system reset password
    Old password:
    New password:
    New password confirmation:
    ⚠️ 输入旧密码错误会造成修改密码失败，失败信息会在整个流程后报出。
    ⚠️ 修改密码需要在重复确认一次。重复确认的密码若于第一次输入的密码不一致，修改密码失败。

    19. 恢复出厂设置
    （此功能已在 FusionStudent Platform lite 4.5.0 版本验证通过。）
    恢复出厂设置，删除环境中存在的所有数据（实例和日志），并不保存数据关机。
    命令输入后会弹出操作确认提示，输入YES后在5秒后系统开始恢复出厂设置。
    该操作属于危险操作，请谨慎使用。
    >>> clear system
    WARNING: It's a dangerous operation. Are you sure you still want to do this operation?(YES/NO) YES
    System will be clear in 5 seconds.
    Delete class 'W97OZ2R9-5NPD-OK4X-O2YU-7VIWKH7N' success.
    System clean up success. It will shutdown in 5 seconds.
    SHUTDOWN
    System run 111.56 seconds.
    ⚠️ 如果环境中存在Lock状态的class，恢复出厂设置失败。
